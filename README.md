# ai-template

All skills, conventions, and AI template files required to bootstrap any of my repositories.

Repositories cloned from this template inherit a `CLAUDE.md` that automatically enforces project conventions — test coverage, directory structure, Makefile targets, and the PROVENANCE origin story — without the user having to ask.

## What's Included

- **[CLAUDE.md](CLAUDE.md)** — Project conventions automatically loaded by Claude Code (README requirements, Makefile targets, test coverage, directory structure)
- **[skills/](skills/)** — Reusable AI prompt templates. See the [Skills Index](skills/README.md) for available skills.

## Usage

1. Clone or use this repo as a GitHub template for a new project.
2. The `CLAUDE.md` will automatically guide AI assistants to follow project conventions.
3. Replace this `README.md` with a project-specific one (the `CLAUDE.md` will remind the AI to do this).
